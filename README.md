## 🌟 Diferenciais do Projeto

- **Branding Sofisticado**: Identidade visual baseada em tons de ameixa profunda e dourado metálico.
- **Extreme Mobile Optimization**: Experiência de navegação fluida em smartphones com menu burger funcional.
- **Integração de Mídia**: Hero section com fundo de vídeo cinematográfico e galeria dinâmica de transformações.
- **Localização em Tempo Real**: Integração com Google Maps para fácil localização em Itaberaba - BA.
- **Ordem de Chegada**: Comunicação clara da política de atendimento direto.

---

## 🎨 Design & Interface

O projeto utiliza **Glassmorphism**, tipografia de alta costura (**Playfair Display**) e animações controladas por **Intersection Observer** para garantir que cada seção do site surja com elegância conforme o usuário navega.

---

## 🚀 Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/SnyX/Atelier-da-Beleza.git
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd Atelier-da-Beleza
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
